## Overview

**Paws & Preferences** is a playful, interactive web application that lets users swipe through adorable cat images and mark them as "Liked" or "Disliked." Inspired by popular swipe-based interfaces, the app provides a fun and engaging way to explore cute kitties online.

This project is fully front-end and relies on HTML, CSS, and JavaScript. It dynamically fetches cat images from [Cataas](https://cataas.com/) and provides an animated, interactive user interface.

---

## Features

- **Swipe Cards:**  
  Swipe right to like a cat and left to dislike, with smooth animations and real-time visual feedback.

- **Audio Feedback:**  
  - "Meow" sound for liking a cat  
  - "Pop" sound for disliking a cat  
  (Implemented using HTML `<audio>` and the Web Audio API)

- **Dynamic Cat Images:**  
  Cat images are loaded dynamically from Cataas with unique timestamps to prevent caching.

- **Summary Panel:**  
  After finishing the deck, a summary panel displays all cats you liked.

- **Responsive Design:**  
  Works well on desktop and mobile screens, with adaptive layouts.

- **Cute Visual Effects:**  
  - Floating sparkles background  
  - Gradient overlays and soft shadows  
  - Animated badges for like/dislike feedback

- **Reset Button:**  
  Reloads the deck to start over.
