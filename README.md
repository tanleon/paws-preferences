# paws-preferences

Small single-page app for discovering which cats you like by swiping.

Notes
- Uses Cataas for cat images.
- Images are lazy-loaded to improve initial load speed.
- Built with vanilla JS and CSS for portability and simplicity.

Deploy
- To host on GitHub Pages: push this repo to GitHub and enable Pages from the repository settings (serve from the main branch / root).
- After publishing, the site will be available at https://<your-username>.github.io/<repo-name>.

Accessibility & controls
- Swipe (touch) or drag (mouse) to like/dislike.
- Buttons available for like/dislike, undo and reset.
- Keyboard: ArrowLeft / ArrowRight and 'U' to undo.
